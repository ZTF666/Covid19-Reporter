# Covid-19 Reporter

A simple webapp that consumes the 'https://covid2019-api.herokuapp.com/v2/' api 👁️ and displays the details .
the app is powered by chartJs 📊, datatables 🧾 , bootstrap , javascript , jquery and ajax.
